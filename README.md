# kiratan-demo

Kawaii-themed AI chat demo for Arigatou Kawaii AI Aishiteru LLC. Pastel UI with floating emoji particles and streaming responses.

## Tech Stack

- Next.js 14, React 18, TypeScript
- Moonshot API (moonshot-v1-8k) for chat completions
- Vercel deployment

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

- `MOONSHOT_API_KEY` -- Moonshot API key
